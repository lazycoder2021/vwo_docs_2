Should I include SmartCode on each and every page of your website?

No. You need to add VWO SmartCode only on those web pages that have been configured for VWO campaigns to run. In other words, you should include VWO SmartCode only on those web pages whose URLs you have submitted to VWO servers for optimization purposes.

Can I use Synchronous SmartCode?

VWO discourages you from using synchronous SmartCode, as it can have a great impact on your web page's loading time. In fact, Asynchronous SmartCode is displayed as the default option for connecting your website with VWO servers. But if you still feel like going for Synchronous Smartcode, get in touch with the support team and seek their assistance.

Can I place the VWO SmartCode snippet at the bottom of the HTML page?

No. You should always add it at the top between <head> tags. And if you place it at the bottom of the page, there will be a huge delay in connecting with VWO servers which can in turn have a great impact on your web page's load time. So always add them at the top.

Can I Customize VWO SmartCode?

Yes, to a certain extent. For example, you can modify the page loading time tolerance (settings_tolerance attribute within SmartCode) of VWO from its default 2000 milliseconds to a value that suits your requirements. You can also allow jquery within your website to be used by setting use_existing_jquery attribute to true. For more information on this, please contact the support team.

What if I don't know the technical know how of installing VWO SmartCode?

Don't worry. You can make use of the 'Send to Developer' button right above the SmartCode snippet and mail your VWO SmartCode to a developer so that he/she can install it for you. You don't even have to give any explicit instructions to them. The info in VWO's mail would be enough to get them to complete the installation process.
